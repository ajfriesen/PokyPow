# Documentation

!!! failure "You need to flash the PokyPow"
    
Do not worry, everything will be done in the browser (Edge, Chrome are supported).
No complex tools are needed.

![flashing.gif](../img/flashing.GIF)

1. Connect your PokyPow via USB-C to your computer/laptop
2. Install drivers (CH340)
3. Click Connect
4. Choose the device CH340 in menu
5. Hold the boot button on the board
6. Click "Install PokyPow"
7. Wait. Keep the Window focused otherwise the install will be slowed down. 
8. Enter your WiFi credentials
9. Have fun!
