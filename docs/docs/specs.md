# Technical Specification

- Microcontroller: ESP32-C3-MINI-1U
- Firmware: ESPHome
- Wireless: 2.4 GHz Wireless LAN + Bluetooth 5 (LE)
- Power Input: 5V via Internal USB Header or USB-C
- Connectors: Pins for Power, Reset and LED for the PC case and motherboard
- External Antenna Connector: I-PEX MHF3, W.FL
- Dimensions: xxmm X xxmm X xxmm
